The goal of our project was to create a soccer ball launcher which would orient itself toward a goal using computer vision.
This is an interesting project because it will help train Reinforcement Learning algorithms for a quadrupedal robot goalkeeper:
https://www.youtube.com/watch?v=iX6OgG67-ZQ.
It is hard to train the goalie because soccer balls that are kicked by a human vary a lot in trajectory. Our robot will be able to accurately reproduce any trajectory so the goalie has ample training data. Current soccer ball launchers cannot orient themselves to precise angles, so our device provides an edge in training time.
